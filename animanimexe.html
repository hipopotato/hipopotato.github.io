AniManim (Versão Desktop .exe)

O que é o AniManim Desktop?

AniManim é uma ferramenta educacional criada para simplificar o uso da poderosa biblioteca de animação **Manim**, do Python. O projeto nasceu como parte de uma dissertação de mestrado do PROFMAT, com o objetivo de tornar a criação de animações matemáticas acessível a professores e alunos.

Esta é a **versão executável para Windows (.exe)**. Você baixa o aplicativo e o executa diretamente no seu computador, sem precisar de um navegador ou conexão com a internet (após a instalação dos requisitos).

Qual é o objetivo?

O aplicativo funciona como uma interface gráfica (GUI) para o Manim. Ele permite que você cole seu código, escolha a qualidade, renderize o vídeo e veja os erros em um console amigável, tudo sem precisar usar o terminal de comando.

Uma das principais ideias deste trabalho é facilitar o uso do Manim para quem não é programador. Como o Manim exige conhecimento de Python, uma **IA (como o Gemini)** pode ser usada para gerar o script da animação. O usuário pode, então, simplesmente **copiar o código da IA e colá-lo no AniManim** para renderizar o vídeo.

O que é o arquivo .exe?

O arquivo `.exe` (que significa "executável") é o aplicativo AniManim completo, "empacotado" para rodar no Windows **sem que você precise instalar o Python ou a biblioteca Manim manualmente**.

Ao baixar o arquivo `.zip`, você encontrará uma pasta contendo o `AniManim.exe` e vários outros arquivos e subpastas. O `.exe` precisa desses arquivos para funcionar, então **mantenha todos os arquivos juntos na mesma pasta** e execute o `AniManim.exe`.

É seguro? Como é um projeto de pesquisa e não um software de uma grande empresa, o Windows ou seu antivírus podem exibir um aviso de "Editor Desconhecido" (SmartScreen). Se isso acontecer, você pode precisar clicar em "Mais informações" e depois em "Executar assim mesmo" para abri-lo.

⚠️ Requisitos Obrigatórios para Rodar

Este aplicativo **NÃO** é um pacote "tudo-em-um". Ele é uma interface que *controla* o Manim. Para que o AniManim funcione, seu computador (Windows) **DEVE TER** os seguintes programas externos instalados e configurados:

1. $\LaTeX$ (Obrigatório)

Essencial para o Manim renderizar qualquer texto ou fórmula matemática.

Recomendação para Windows: MiKTeX (Baixe aqui)

2. FFmpeg (Obrigatório)

Fundamental para o Manim juntar as imagens e criar o arquivo de vídeo `.mp4`.

Link para Download: FFmpeg.org (Baixe aqui)

Dica: Após instalar ambos, abra o AniManim, clique no botão **'i' (Informações)** e use o botão **"Verificar Dependências"** para checar se o aplicativo está conseguindo encontrá-los no seu sistema.

Instruções de Instalação

  1. Faça o download do arquivo `AniManim.zip` usando o botão no topo da página.
  2. Clique com o botão direito no arquivo `.zip` baixado e escolha **"Extrair tudo..."**.
  3. Extraia os arquivos para uma pasta de sua preferência.
  4. Abra a pasta `AniManim` que foi criada.
  5. Clique duas vezes em `AniManim.exe` para iniciar o programa.
  6. Não se esqueça de ler o arquivo `LEIA-ME.txt` (incluído no ZIP) para instruções completas!

O Processo de Criação (Professora + IA)

Este projeto é uma colaboração direta entre uma educadora e uma inteligência artificial. A visão do projeto, os requisitos educacionais, a direção do fluxo de trabalho e o teste de funcionalidade foram fornecidos por mim, uma professora de matemática em formação no PROFMAT. O código-fonte completo da interface (Python/Tkinter) foi gerado, depurado e refinado pelo **Gemini**, uma IA conversacional do Google.

Aviso Final

Este é um projeto experimental e de prova de conceito. Ele pode conter erros. Como mencionado, foi realizado com o auxílio de IA por uma professora de matemática, não por uma programadora profissional.
